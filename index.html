GitHub Pages

Serve para hospedar uma página na internet, gerando uma url para ser enviado para alguém que não tem acesso ao PC onde a página foi criada.

Para criar um link, siga os passos:

  1. Entre na página do repositório desejado;

  2. Clique Settings, que está no menu na parte superior da página;

  3. Clique em Pages que está no menu da direita na guia Code and automation;

  4. Já na seção GitHub Pages localize o Branch e altere de None para main;

  5. Ao lado do main a opção root deve estar selecionada;

  6. Clique em Save.

Pronto, a página ja foi criada.

Para ter acesso ao link gerado nos passos anteriores:

  1. Volte até a raiz do repositório;

  2. No menu da direita, na guia Environments, clique em github-pages;

  3. Identifique onde está marcado o tempo de criação do arquivo, exemplo: 3 minutes ago e clique na caixa com uma seta sainda dela;

  4. Agora é só copiar o link que está na barra de endereço da página.
